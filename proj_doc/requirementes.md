## 目的

VRC ワールド情報を DB で管理し，名前・タグで検索できる API を提供する
PC よりローカルサーバを建てることにより，Quest 単騎でも使える WorldStorage を作る

## スコープ

インスコープ
DB に保存されたワールドを tag 検索する
VRCAPI でワールド検索を行う
新規 DB 保存を行う
DB の編集を行う
編集 DB の更新を行う
DB 保存を削除する
将来拡張
VRCAPI 自動ログインのメンテ用 API を作成する
最近更新したワールド 10 件を知らせる recent の作成

α. 背景
何らかの PC で鯖を建てることにより，quest 単騎でも扱える worldstorage アプリを作る
β. 概要
開発体制: 一人
開発方式: MVC 開発，ボトムアップアプローチで要件定義を行う. issue 駆動を行う．
